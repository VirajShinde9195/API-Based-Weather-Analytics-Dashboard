🌦️ API-Based Weather Analytics Dashboard
📌 Project Overview

This project demonstrates an end-to-end data analytics pipeline using real-time weather data fetched from a REST API. It focuses on collecting, cleaning, analyzing, and visualizing temperature data across multiple cities to generate meaningful insights.

The project simulates a real-world workflow where raw API data is transformed into actionable insights for data-driven decision-making.

🚀 Project Workflow
1. Data Extraction
Fetched real-time weather data using REST API
Parsed nested JSON data using Python
2. Data Processing
Converted raw JSON into structured format using Pandas
Cleaned and transformed data for analysis
3. Exploratory Data Analysis (EDA)
Identified temperature trends and patterns
Compared variations across cities and time
4. Visualization
Created charts using Matplotlib
Built an interactive dashboard using Power BI
5. Insights Generation
Derived key findings on temperature behavior
Compared city-wise climate patterns
📊 Dashboard Features
Temperature Trend Over Time (Line Chart)
Average Temperature by City (Bar Chart)
KPI Cards (Avg, Max, Min Temperature)
Interactive Filters (City & Time Selection)
Detailed Data Table
